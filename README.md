# Task Manager

A simple Task Manager web application built using Laravel and MySQL.

## Features

- Create Task
- Edit Task
- Delete Task
- Task Status (Pending / Completed)
- Dashboard Statistics
- Task Search
- Bootstrap Responsive UI
- Delete Confirmation Popup

## Technologies Used

- Laravel
- PHP
- MySQL
- Bootstrap

## Installation

1. Clone the repository

git clone https://github.com/Naweju12/Task_Manager.git

2. Navigate to project folder

cd task_manager

3. Install dependencies

composer install

4. Configure database in `.env`

5. Run migrations

php artisan migrate

6. Start server

php artisan serve
